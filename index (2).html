Command History:

    
    
    

    
    
        
    Command Definitions
    

        
        
    
            
    INIT
    
            
    The 'init' command is used to initialize the system and manage run levels. It is responsible for starting system processes during boot-up.
    
            
    
                Usage:
                
    init [level]
    
            
    
            
    
                Example:
                
    init 3 - Switches to multi-user mode without a graphical interface.
    
            
    
            
        
    

        
        
    
            
    USERADD
    
            
    The 'useradd' command is used to create a new user on the system, allowing them to have login privileges and access to the system.
    
            
    
                Usage:
                
    useradd [options] username
    
            
    
            
    
                Example:
                
    useradd -m john - Creates a new user 'john' with a home directory.
    
            
    
            
        
    

        
        
    
            
    USERMOD
    
            
    The 'usermod' command allows modification of user account details, such as username, home directory, and group memberships.
    
            
    
                Usage:
                
    usermod [options] username
    
            
    
            
    
                Example:
                
    usermod -aG sudo john - Adds user 'john' to the 'sudo' group.
    
            
    
            
        
    

        
        
    
            
    USERDEL
    
            
    The 'userdel' command removes a user from the system, optionally deleting their home directory as well.
    
            
    
                Usage:
                
    userdel [options] username
    
            
    
            
    
                Example:
                
    userdel -r john - Deletes user 'john' along with their home directory.
    
            
    
            
        
    

        
        
    
            
    GROUPADD
    
            
    The 'groupadd' command creates a new group on the system.
    
            
    
                Usage:
                
    groupadd groupname
    
            
    
            
    
                Example:
                
    groupadd developers - Creates a new group called 'developers'.
    
            
    
            
        
    

        
        
    
            
    GROUPMOD
    
            
    The 'groupmod' command allows modifications to an existing group, such as changing its name or GID.
    
            
    
                Usage:
                
    groupmod [options] groupname
    
            
    
            
    
                Example:
                
    groupmod -n newgroup developers - Renames the 'developers' group to 'newgroup'.
    
            
    
            
        
    

        
        
    
            
    GROUPDEL
    
            
    The 'groupdel' command deletes a group from the system.
    
            
    
                Usage:
                
    groupdel groupname
    
            
    
            
    
                Example:
                
    groupdel developers - Deletes the 'developers' group from the system.
    
            
    
            
        
    

        
        
    
            
    PASSWD
    
            
    The 'passwd' command allows the system administrator or a user to change their password.
    
            
    
                Usage:
                
    passwd [options] username
    
            
    
            
    
                Example:
                
    passwd john - Changes the password for the user 'john'.
    
            
    
            
        
    

        
        
    
            
    SUDO
    
            
    The 'sudo' command grants administrative privileges to users, allowing them to execute commands with superuser rights.
    
            
    
                Usage:
                
    sudo command
    
            
    
            
    
                Example:
                
    sudo apt update - Runs the 'apt update' command with administrative privileges.
